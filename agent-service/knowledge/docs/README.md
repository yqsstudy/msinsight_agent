# 知识文档目录

将调优知识文档放在此目录下，支持以下格式：
- `.md` - Markdown文档
- `.txt` - 纯文本文档
- `.rst` - reStructuredText文档

## 文档结构建议

```
docs/
├── communication_optimization.md   # 通信优化指南
├── memory_optimization.md          # 内存优化指南
├── performance_tuning.md           # 性能调优指南
├── troubleshooting.md              # 问题排查手册
└── best_practices.md               # 最佳实践
```

## 文档内容建议

每个文档应包含：
1. 问题类型描述
2. 常见原因分析
3. 诊断方法
4. 解决方案
5. 案例示例
